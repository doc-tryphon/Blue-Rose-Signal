## Active Platform: [Blue Rose Signal Next](/blue-rose-signal-next)

The active system is a **monorepo-style Next.js 16 application** serving three distinct operational modules. It enforces a strict **Vintage CRT Aesthetic** (Amber Monochrome, Scanlines) to maintain containment protocols.

### System Modules

#### 1. [SIGNAL SPECTROGRAPH](blue-rose-signal-next/SPECTROGRAPH_MANUAL.md)
*   **Designation**: `PUBLIC UTILITY`
*   **Function**: Real-time stochastic interference synthesis ("The Hum").
*   **Core Tech**: Web Audio API AudioWorklet, Box-Muller Gaussian distribution.
*   **Capabilities**:
    *   60Hz Harmonics Control
    *   Ungated Spark Gap Simulation (Friction-based)
    *   Ring Modulation ("Ghost in the Radio")
    *   Voltage Starvation Visualization

#### 2. [PROJECT AEON](/aeon) (Neural Interface)
*   **Designation**: `CLASSIFIED // EYES ONLY`
*   **Function**: Secure terminal for communicating with the AEON Large Language Entity.
*   **Core Tech**: Markdown/LaTeX Rendering, Real-time Streaming.
*   **Capabilities**:
    *   **ChatInterface**: Terminal-style log with math proof validation.
    *   **FidelityMeter**: Visualizing signal integrity/hallucination rates.
    *   **ProtocolState**: Monitoring containment stability.

#### 3. [MAINFRAME DASHBOARD](/mainframe)
*   **Designation**: `AUTHORIZED PERSONNEL`
*   **Function**: Central command for Division operations and case management.
*   **Core Tech**: Next.js App Router, Dynamic Routing.
*   **Capabilities**:
    *   **System Status**: Real-time diagnostics (Entropy Index, Defcon Status).
    *   **Active Protocols**: Operational mandates and sub-routines.
    *   **Case Archive**: Access to classified anomaly files (e.g., Case 893 "The Hiss").

## Quick Start

To initialize the Blue Rose Systems terminal:

```bash
cd blue-rose-signal-next
npm install
npm run dev
```

Access the **Boot Menu** at `http://localhost:3000`.
